# Sleep Tracker Application

## Description
This is a Sleep Tracker application that helps users record and view their sleep data using a simple dashboard.

## Features
- Sleep dashboard
- View sleep records
- Simple frontend UI
- Basic backend using Node.js

## Tech Stack
- Frontend: HTML, CSS, JavaScript, React
- Backend: Node.js

## How to Run
1. Clone the repository
2. Install dependencies using `npm install`
3. Start the project using `npm start`
